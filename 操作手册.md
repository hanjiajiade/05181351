## 一、本地预览

### 方式一：直接打开（最简单）

1. 找到项目文件夹中的 `aura-product-page.html` 文件
2. 双击该文件，系统默认浏览器会自动打开
3. 即可查看完整网页效果

> **注意**：直接打开时，部分浏览器的安全策略可能会限制CSS动画或字体加载，建议使用方式二获得最佳体验。

### 方式二：本地服务器（推荐）

#### Windows 系统

**使用 Python（如已安装）：**
```
1. 按 Win+R，输入 cmd，回车打开命令行
2. cd 进入项目文件夹目录，例如：
   cd C:\Users\你的用户名\Desktop\Aura-Product-Page
3. 执行命令：
   python -m http.server 8080
4. 打开浏览器，访问：http://localhost:8080/aura-product-page.html
```

**使用 VS Code：**
```
1. 用 VS Code 打开项目文件夹
2. 安装 "Live Server" 插件
3. 右键点击 aura-product-page.html → "Open with Live Server"
```

#### macOS 系统

**使用终端：**
```
1. 打开「终端」应用
2. cd 进入项目文件夹目录，例如：
   cd ~/Desktop/Aura-Product-Page
3. 执行命令：
   python3 -m http.server 8080
4. 打开浏览器，访问：http://localhost:8080/aura-product-page.html
```

#### Linux 系统

```
1. 打开终端
2. cd 进入项目文件夹目录
3. 执行命令：
   python3 -m http.server 8080
4. 打开浏览器，访问：http://localhost:8080/aura-product-page.html
```

## 二、服务器部署

### 方式一：静态文件托管（推荐）

适用于 Nginx / Apache 等Web服务器。

#### Nginx 配置示例

```nginx
server {
    listen 80;
    server_name your-domain.com;

    root /var/www/aura;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }

    # 图片资源缓存
    location ~* \.(jpg|jpeg|png|gif|ico|svg|webp)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }

    # Gzip 压缩
    gzip on;
    gzip_types text/html text/css application/javascript image/svg+xml;
}
```

**部署步骤：**
```bash
# 1. 将项目文件上传到服务器
scp -r ./Aura-Product-Page/* user@server:/var/www/aura/

# 2. 重启 Nginx
sudo nginx -s reload
```

#### Apache 配置示例

将项目文件放入 Apache 的 `htdocs` 或 `DocumentRoot` 目录即可：

```bash
cp -r ./Aura-Product-Page/* /var/www/html/
```

### 方式二：云平台部署

#### 部署到 Vercel

```bash
# 1. 安装 Vercel CLI
npm i -g vercel

# 2. 在项目目录下执行
cd Aura-Product-Page
vercel

# 3. 按提示操作，部署完成后获得访问链接
```

#### 部署到 GitHub Pages

```
1. 将项目上传到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. Source 选择 "Deploy from a branch"
4. Branch 选择 main，文件夹选 / (root)
5. 点击 Save，等待部署完成
6. 通过 https://yourusername.github.io/repo-name/aura-product-page.html 访问
```

#### 部署到阿里云 OSS / 腾讯云 COS

```
1. 登录云服务控制台
2. 创建存储桶（Bucket），权限设为「公共读」
3. 上传所有项目文件
4. 绑定自定义域名（可选）
5. 开启 CDN 加速（可选）
```

### 修改产品信息

打开 `aura-product-page.html`，搜索以下关键词进行修改：

| 需要修改的内容 | 搜索关键词 | 位置说明 |
|---------------|-----------|---------|
| 产品名称 | `Aura` | 导航栏、Hero标题 |
| 产品副标题 | `AI蓝牙耳机` | Hero副标题区域 |

| AI功能列表 | `语音控制` 等 | AI助手区域 |

### 替换产品图片

1. 准备新图片，建议尺寸：
   - Hero主视觉：1280 x 720 px
   - 场景图：1280 x 720 px
   - AI展示图：800 x 800 px

2. 将新图片放入 `assets/` 文件夹，替换对应文件：
   - `hero-product-visual.jpg` → Hero区域主视觉
   - `ai-assistant-visual.jpg` → AI助手展示图
   - `scene-commute.jpg` → 通勤场景
   - `scene-sport.jpg` → 运动场景
   - `scene-work.jpg` → 工作场景

3. 保持文件名一致，或同步修改HTML中的图片路径

### 修改配色方案

在HTML文件的 `<style>` 标签中搜索以下颜色值：

```css
--color-primary: #bf5af2;      /* 主色调（紫色） */
--color-secondary: #2997ff;    /* 辅助色（蓝色） */
--color-bg: #000000;           /* 背景色（黑色） */
--color-text: #f5f5f7;         /* 主文字色（白色） */
```

### 修改导航链接

搜索 `<nav>` 标签，修改 `<a href="#xxx">` 中的锚点链接和文字。

## 四、常见问题

### Q：图片无法显示？
- 检查 `assets/` 文件夹是否与 `aura-product-page.html` 在同一目录
- 确认图片文件名与HTML中引用的路径一致
- 如使用服务器部署，检查文件权限

### Q：动画效果不流畅？
- 确保使用现代浏览器（Chrome 80+、Safari 14+）
- 检查浏览器是否开启了「减少动画」模式
- 关闭不必要的浏览器插件

### Q：手机端显示异常？
- 本页面已做响应式适配，如仍有问题请清除浏览器缓存后重试
- 建议使用系统自带浏览器访问

### Q：如何修改页面布局？
- 页面采用 Flexbox + Grid 布局
- 修改 `<style>` 标签中的对应CSS即可
- 建议具备基本的HTML/CSS知识
