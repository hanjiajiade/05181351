# Aura AI蓝牙耳机 - 产品网页

> 沃垠科技出品 | 重新定义无线聆听体验

## 项目简介

Aura 是沃垠科技推出的AI蓝牙耳机产品官网页面，采用苹果风格的高端设计语言，融合顶尖降噪技术与智能语音助手的展示，带来沉浸式的产品浏览体验。

## 项目结构

```
Aura-Product-Page/
├── aura-product-page.html      # 主页面文件（包含所有HTML/CSS/JS）
├── assets/                     # 图片资源目录
│   ├── hero-product-visual.jpg    # Hero区域产品主视觉图
│   ├── ai-assistant-visual.jpg    # AI智能助手功能展示图
│   ├── scene-commute.jpg          # 通勤场景图
│   ├── scene-sport.jpg            # 运动健身场景图
│   └── scene-work.jpg             # 专注工作场景图
├── README.md                   # 项目说明文件（本文件）
└── 操作手册.md                  # 部署与操作手册
```

## 技术栈

- **纯前端实现**：HTML5 + CSS3 + JavaScript（原生）
- **无需构建工具**：无需 Node.js、npm 或任何框架
- **响应式设计**：适配桌面端、平板端、移动端
- **浏览器兼容**：Chrome 80+、Firefox 80+、Safari 14+、Edge 80+

## 设计特点

| 特性 | 说明 |
|------|------|
| 配色方案 | 紫色(#bf5af2) + 蓝色(#2997ff) 双色调 |
| 设计风格 | 参考苹果 AirPods Pro 官网 |
| 动画效果 | 入场动画 + 滚动淡入 + 悬停交互 |
| 图标系统 | 精致SVG内联图标 |
| 导航栏 | 毛玻璃效果固定导航 |
| 字体 | SF Pro风格系统字体栈 |

## 页面模块

1. **导航栏** - 固定顶部，毛玻璃效果
2. **Hero区域** - 产品主视觉 + CTA按钮
3. **核心功能** - 6大功能特性卡片
4. **AI智能助手** - 功能列表 + 可视化展示
5. **使用场景** - 3张AI生成场景图
6. **技术规格** - 核心参数数据展示
7. **购买CTA** - 价格与购买引导
8. **页脚** - 品牌信息与链接

## 快速开始

1. 解压项目文件
2. 双击打开 `aura-product-page.html` 即可在浏览器中预览
3. 或使用本地服务器部署（推荐）

详细部署说明请参阅 [操作手册.md](./操作手册.md)
